# CSC415 - Operating Systems - Group Term Assignment - File System

This assignment was a group project assigned to use by our operating systems professor. We worked in a team of 4 to create a filesystem in linux using no large libraries but basic low level functions provivded by our professor.

## My contribution to the project
- Writing the FAT table from scratch which is a type of free space structure to determine what block are free, so we can write blocks to disk. 
- Few of the shell functions
- Few of the low level read/write functions

## Project Summary / What I learned
This project was the most fun project Ive worked on during my undergraduate studies, I realized how much I enjoy working at a low level and having the opportunity and freedom to work on this big of a project was incredibly enjoyable.
I learned so much about C/C++, linux and how the filesystem and the operating system works at a very low level.
